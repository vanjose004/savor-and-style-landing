# Savor and Style Catering Services — Landing Page

Marketing landing page for Savor and Style Catering Services (V&A Estrella Catering Services). Built with React, Vite, Tailwind CSS, and Framer Motion.

This is a static marketing site — bookings, payments, and client management are handled separately by the [client portal](https://savor-and-style-portal.vercel.app).

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

See [SAVOR_AND_STYLE_LANDING.md](./SAVOR_AND_STYLE_LANDING.md) for the full project blueprint.
